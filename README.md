# Validity Hybrid Tipbot

##### THIS IS EXPERIMENTAL SOFTWARE USE AT YOUR OWN RISK

A tipbot that is interoperable across Telegram and Discord.

## Dependencies

* Firebase credientals + JSON private key
* Telegram bot token
* Discord bot token
* Node.js

## Installation

* Firebase credientals + JSON private key
* Telegram bot token
* Discord bot token
* `node bot.js`

## Commands

